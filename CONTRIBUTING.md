# Contributing to Metashade

Welcome, and thank you for your interest in contributing to Metashade!

## Developer Certificate of Origin (DCO)
To ensure the legal integrity of the codebase, this repository enforces the Developer Certificate of Origin (DCO). All contributions must be signed off to confirm the contributor has the right to submit the code under the project's open-source license.

You can easily sign your commits by adding the `-s` or `--signoff` flag to your git commit command:
`git commit -s -m "Your commit message"`

If you forgot to sign your last commit, you can fix it by running:
`git commit --amend --no-edit --signoff`
followed by `git push --force-with-lease`.
